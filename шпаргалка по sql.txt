Типы данных в sql:
INT	Целое число	id пользователя: 132
NUMERIC / DECIMAL - Вещественное число 120.55
VARCHAR - Текст
DATE - Дата с точностью до дня	
TIMESTAMP - Дата с точностью до секунды
[] - Массив(array)


Правильный порядок sql запросов
SELECT -- перечисление полей результирующей таблицы
FROM -- указание источника данных
WHERE -- фильтрация данных
GROUP BY -- группировка данных
HAVING -- фильтрация данных после группировки
ORDER BY -- сортировка результирующей таблицы
LIMIT -- ограничение количества выводимых записей

В то же время порядок выполнения операторов в запросе следующий:

Сначала выполняется оператор FROM — происходит выбор нужной таблицы.
FROM       -- указание источника данных
WHERE      -- фильтрация данных
GROUP BY   -- группировка данных
HAVING     -- фильтрация данных после группировки
SELECT     -- перечисление полей результирующей таблицы
ORDER BY   -- сортировка результирующей таблицы
LIMIT      -- ограничение количества выводимых записей

Математические операторы:
+, -, *, /, % - остаток от деления, ^ - возведение в степень, |/ 25 - изввлечение квадратного корня,
||/ - извлечение кубического корня, 5 ! - факториал, @ -5.0 - абсолютное значение, 


К операторам сравнения относятся:
= («равно»)
<> или != («не равно»)
< («меньше»)
> («больше»)
<= («меньше или равно»)
>= («больше или равно»)
!< - не меньше
!> - не больше
BETWEEN - вхождение в диапазон
IS NULL - Значение Null


С этими тремя состояниями можно проводить следующие логические операции:
AND («И»)
OR («ИЛИ»)
NOT («НЕ») - ставится перед условием

Оформление даты: 'year/month/day'
timestamp, currennt_date - сегодняшняя дата
