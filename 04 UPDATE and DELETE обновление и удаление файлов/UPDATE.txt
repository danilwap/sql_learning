Есть два способа использования UPDATE
- Обновление определённых строк в таблице
UDATE Customers
SET cust_email = 'bes@bs.ru',
	cust_name = 'Danila'
WHERE cust_id = '100000005'
- Обновление всех строк в таблице

Состовляющие инструкции:
- Имя таблицы, подлежащей обновлению;
UDATE Customers
- Имена столбцов и их новые значения;
SET cust_email = 'value'
- Условия фильтрации, определяющие, какие именно строки должны быть обновлены.
WHERE cust_id = 123232323
