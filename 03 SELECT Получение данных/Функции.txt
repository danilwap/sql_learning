Для извлечения даты из TIMESTAMP можно использовать:
DATE(current_time)
CAST(time AS DATE)
time::DATE

Для объединения строк используется CONCAT
CONCAT(first_st, second_st)

Для получения части timestamp есть функция, part = 'year', 'day', 'month', 'dow' - день недели(воскресенье - 0, понедельник -1, и т.д.),
'hours', 'minute'
SELECT DATE_PART(part, column)
DATE_TRUNC('year', date) - округление даты
TO_CHAR(date, 'Dy') - возвращает название дня недели

AGE(date1, date2) - разница между датами
AGE(current_date, date2) - разница между датой и сегодняшней датой
AGE(TIMESTAMP date2) - разница между датой и сегодняшней датой
AGE(current_date, date2)::VARCHAR - разница между датой и сегодняшней датой, представить в веде 1 year, 2 mon и т.д.

Для того, что поле не было NONE есть функция COALESCE
COALESCE(DATE_PART('year', birth_date)::varchar, 'unknown') # Значение обоих значений должно быть одного формата

Некоторые математические функции функции
AVG() - среднее
ABS(-10) = 10 - модуль числа
round(2.2222, 2) = округление до указанного количества знаков
CEIL(1.2) = 2 - округления до ближайшего большего числа
FLOOR(10.9) - округление до меньшего 
SCALE(2.2323) - количество знаков после запятой
RANDOM() - генерирует случайное число от 0 до 1
